# Resumo de Química Orgânica - Cesgranrio (~5 questões)

---

## 1. HIDROCARBONETOS E PETRÓLEO

### Classificação das Cadeias Carbônicas

| Critério | Tipos |
|----------|-------|
| Fecho | Aberta (acíclica) | Fechada (cíclica) |
| Ramificação | Normal | Ramificada |
| Ligação | Saturada (simples) | Insaturada (dupla/tripla) |
| Natureza | Homogênea (só C) | Heterogênea (outro átomo) |

### Hidrocarbonetos

| Família | Fórmula geral | Ligação | Exemplo |
|---------|--------------|---------|---------|
| Alcantoalcano | CₙH₂ₙ₊₂ | simples | CH₃-CH₃ (etano) |
| Alcenoteno | CₙH₂ₙ | 1 dupla | CH₂=CH₂ (eteno) |
| Alcinotino | CₙH₂ₙ₋₂ | 1 tripla | CH≡CH (etino) |
| Alcadieieno | CₙH₂ₙ₋₂ | 2 duplas | CH₂=C=CH₂ |
| Cicloalcano | CₙH₂ₙ | fechada simples | C₆H₁₂ (cicloexano) |

### Petróleo e Refino
- **Destilação fracionada:** Separa por ponto de ebulição
- **Frações (do mais leve ao mais pesado):**
  1. Gás (metano, etano)
  2. GLP (propano, butano) 
  3. Gasolina (C₅-C₁₀)
  4. Querosene (C₁₁-C₁₃)
  5. Diesel (C₁₄-C₁₈)
  6. Óleo lubrificante
  7. Asfalto

**Macete Cesgranrio:** Quanto MAIOR a cadeia, MAIOR o ponto de ebulição. Quanto MAIS ramificada, MENOR o ponto de ebulição.

## 2. NOMENCLATURA IUPAC

**Regra:** Prefixo (quantidade de C) + infixo (tipo de ligação) + sufixo (função)

| C | Prefixo |
|---|---------|
| 1 | met- |
| 2 | et- |
| 3 | prop- |
| 4 | but- |
| 5 | pent- |
| 6 | hex- |
| 7 | hept- |
| 8 | oct- |
| 9 | non- |
| 10 | dec- |

| Ligação | Infixo |
|---------|--------|
| Simples | -an- |
| Dupla | -en- |
| Tripla | -in- |

**Exemplo:** CH₃-CH₂-CH₃ = prop**an**o. CH₃-CH=CH₂ = prop**en**o.

**Regra para ramificações:** Numerar a cadeia principal para dar o menor número possível às insaturações/ramificações.

## 3. FUNÇÕES ORGÂNICAS

| Função | Grupo | Exemplo | Sufixo |
|--------|-------|---------|--------|
| **Álcool** | -OH | CH₃-OH (metanol) | -ol |
| **Fenol** | -OH no anel | C₆H₅-OH | -ol |
| **Éter** | -O- | CH₃-O-CH₃ | -óxi + ... + ano |
| **Aldeído** | -CHO | CH₃-CHO (etanal) | -al |
| **Cetona** | -CO- | CH₃-CO-CH₃ (propanona) | -ona |
| **Ácido carboxílico** | -COOH | CH₃-COOH (ácido etanoico) | -oico |
| **Éster** | -COO- | CH₃-COO-CH₃ | -ato de ... + ila |
| **Amina** | -NH₂ | CH₃-NH₂ | -amina |
| **Amida** | -CONH₂ | CH₃-CONH₂ | -amida |

**Macete Cesgranrio:** Saber RECONHECER a função pelo grupo é o que mais cai. Não precisa decorar todos os nomes.

## 4. ISOMERIA

### Isomeria Plana
| Tipo | Exemplo |
|------|---------|
| Cadeia | butano (linear) × isobutano (ramificado) |
| Posição | 1-butanol × 2-butanol |
| Função | álcool × éter (C₂H₆O) |
| Metameria | éter com O em posição diferente |
| Tautomeria | enol × cetona (aldeído) |

### Isomeria Geométrica (cis-trans)
- Ocorre em alcenos com ligantes diferentes em cada carbono da dupla
- **cis:** iguais do mesmo lado
- **trans:** iguais em lados opostos

### GABARITO COMENTADO

**Q1:** C (só ligações simples entre carbonos = alcano)
**Q2:** C (4 carbonos = but, dupla entre C2 e C3 = 2-eno)
**Q3:** C (OH ligado a carbono saturado = álcool)
**Q4:** B (gasolina: C₅-C₁₀ ≈ 40-175°C)
**Q5:** C (mesma fórmula, diferentes arranjos da cadeia)
**Q6:** B (ácido + álcool → éster + água = esterificação)
**Q7:** C (cadeia principal: 4 carbonos, metil no C2)
**Q8:** B (C=O entre dois carbonos = cetona)
**Q9:** C (anel benzênico = aromático)
**Q10:** C (3 carbonos = prop, ácido = -oico → propanoico)
