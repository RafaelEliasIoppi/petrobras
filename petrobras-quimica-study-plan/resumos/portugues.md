## 1. INTERPRETAÇÃO DE TEXTOS (~5 questões)

**Regra de ouro Cesgranrio:** A resposta está SEMPRE no texto. Não responda com sua opinião.

| Tipo de questão | O que fazer |
|----------------|-------------|
| Ideia principal | Pergunte: "Sobre o que o texto fala?" |
| Inferência | O texto sugere, mas não diz explicitamente |
| Vocabulário | Volte ao texto, veja o contexto da palavra |
| Tipologia textual | Narrativo (conta história), Descritivo (caracteriza), Dissertativo (defende tese) |

**Macete Cesgranrio:** Leia as perguntas ANTES do texto. Você sabe o que procurar.

## 2. SINTAXE (~3 questões)

### Concordância Verbal

| Regra | Exemplo |
|-------|---------|
| Sujeito + verbo | **Ele** vai. **Eles** vão. |
| Sujeito coletivo (maioria, metade) + de + plural | **A maioria dos alunos foi** OU **foram** (ambas certas) |
| Sujeito + "um dos que" | **Foi um dos que chegou** OU **chegaram** (ambas certas) |
| "Se" + verbo (índice de indeterminação) | **Precisa-se** de funcionários (VTI + SE = singular) |
| "Se" + verbo (partícula apassivadora) | **Vendem-se** casas (VTDI + SE = concorda com sujeito) |

**Macete:** Sujeito antes do verbo → concordância normal. Sujeito depois → pode ficar no singular.

### Concordância Nominal

| Regra | Exemplo |
|-------|---------|
| Adjetivo concorda com substantivo | **Casa bonita**, **carro bonito** |
| Adjetivo depois de dois substantivos | **Homem e mulher bonitos** (plural) |
| Adjetivo antes de dois substantivos | **Bonita casa e jardim** (concorda com o primeiro) |
| É bom, é necessário + sujeito | **É necessário paciência** (invariável com sujeito sem artigo) |
| Anexo, incluso, obrigado | **Anexas** as cópias. **Obrigada** (se mulher fala) |

### Regência Verbal

| Verbo | Regência | Exemplo |
|-------|----------|---------|
| Assistir (ver) | **A** + OI | **Assisti ao filme** |
| Assistir (ajudar) | OD | **Assisti o paciente** |
| Aspirar (cheirar) | **A** + OI | **Aspirei ao perfume** |
| Aspirar (desejar) | **A** + OI | **Aspiro ao cargo** |
| Visar (mirar) | **A** + OI | **Visei ao alvo** |
| Visar (rubricar) | OD | **Visei o documento** |
| Querer (desejar) | OD | **Quero um café** |
| Querer (estimar) | **A** + OI | **Quero bem a você** |
| Pagar / Perdoar | OD + **a** OI | **Paguei o aluguel ao dono** |
| Informar / Avisar | OD + **de** OI | **Informei o cliente do resultado** |

### Crase

**Regra básica:** Crase = **a** + **a** (preposição + artigo).
Usa crase quando o termo exige "a" E o termo seguinte aceita artigo "a".

| Usa crase | Não usa crase |
|-----------|--------------|
| **À** vista (à + a vista) | **A** partir de |
| **À** noite (à + a noite) | **A** pé, **a** cavalo |
| **Às** vezes (às + as vezes) | **A** distância (sem especificar) |
| **À** moda de (à + a moda) | Antes de palavra masculina |
| **Àquela** / **àquele** | Antes de verbo |

**Macete:** Troque por "ao". Se fizer sentido, tem crase.
- "Fui **a**/à praia" → "Fui **ao** mar" ✓ → crase
- "Comecei **a** estudar" → "Comecei **ao** estudar" ✗ → sem crase

## 3. MORFOLOGIA (~1-2 questões)

### Pronomes - Colocação

| Regra | Ênclise (depois) | Próclise (antes) |
|-------|-----------------|-----------------|
| Normal | **Chamou-me** | — |
| Palavra negativa | — | **Não me chamou** |
| Advérbio | — | **Aqui se faz** |
| Pronome relativo | — | **Que me disse** |
| Conjunção | — | **Quando me viu** |
| Início de frase | **Digam-me** | **Me** digam (informal, aceito na fala) |

### Verbos - Vozes

| Voz | Estrutura | Exemplo |
|-----|-----------|---------|
| Ativa | Sujeito **faz** a ação | **O aluno** entregou o trabalho |
| Passiva analítica | Sujeito **recebe** a ação | O trabalho **foi entregado pelo** aluno |
| Passiva sintética | Verbo + **se** | **Entregou-se** o trabalho |

**Dica Cesgranrio:** Transformação ativa ↔ passiva cai frequentemente.

## 4. PONTUAÇÃO (~1 questão)

| Sinal | Uso |
|-------|-----|
| **Vírgula** | Separar termos, isolar explicações, separar orações |
| **Ponto e vírgula** ; | Separar itens de uma lista complexa |
| **Dois-pontos** : | Introduzir citação, explicação, enumeração |
| **Travessão** — | Indicar fala, enfatizar, substituir parênteses |

**Não se separa com vírgula:**
- Sujeito + Verbo
- Verbo + Objeto  
- Nome + Complemento

### GABARITO COMENTADO

**Q1:** D (texto diz "cada fração possui uma faixa de temperatura")
**Q2:** C ("nesse contexto" = na crescente demanda)
**Q3:** D ("Vendem-se" = part. apassivadora, concorda com "apartamentos")
**Q4:** A (Fui **à** praia, voltei **a** tarde, **À** noite, **a** estudar)
**Q5:** C ("Aspirar" no sentido de almejar exige preposição "a")
**Q6:** D (oração explicativa entre vírgulas)
**Q7:** D (advérbio "aqui" exige próclise)
**Q8:** C ("essenciais para garantir que os produtos atendam às especificações")
**Q9:** A (voz passiva: OD vira sujeito + verbo ser + particípio + agente)
**Q10:** B ("é" = verbo ser; "preciso" = adjetivo, qualifica o resultado)
