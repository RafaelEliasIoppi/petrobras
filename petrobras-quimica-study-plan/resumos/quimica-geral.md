# Resumo de Química Geral e Inorgânica - Cesgranrio (~15 questões)

---

## 1. FUNÇÕES INORGÂNICAS (~7 questões) ← MAIS IMPORTANTE

### Ácidos
| Característica | Exemplo |
|----------------|---------|
| Liberam H⁺ em água | HCl → H⁺ + Cl⁻ |
| pH < 7 | |
| Fortes: HCl, H₂SO₄, HNO₃ | Ionizam completamente |
| Fracos: H₂CO₃, H₃PO₄, HCN | Ionizam parcialmente |

**Macete:** Ácido sem oxigênio (H + ametal) → prefixo **hidro** + sufixo **ídrico**.
Ex: HCl = ácido **clorídrico**, H₂S = ácido **sulfídrico**.

**Macete:** Ácido com oxigênio → sufixo:
- **ico** (mais oxigênio): H₂SO₄ = sulfúrico
- **oso** (menos oxigênio): H₂SO₃ = sulfuroso

### Bases
| Característica | Exemplo |
|----------------|---------|
| Liberam OH⁻ em água | NaOH → Na⁺ + OH⁻ |
| pH > 7 | |
| Fortes: NaOH, KOH, Ca(OH)₂ | Metais alcalinos e alcalino-terrosos |
| Fraca: NH₄OH | |

**Nomenclatura:** Hidróxido de + nome do metal.
NaOH = hidróxido de sódio. Ca(OH)₂ = hidróxido de cálcio.

### Sais
| Característica | Exemplo |
|----------------|---------|
| Ácido + Base → Sal + Água | HCl + NaOH → **NaCl** + H₂O |
| Todo sal tem cátion (metal) + ânion (ametal/radical) | NaCl = Na⁺ + Cl⁻ |

**Nomenclatura:** Nome do ânion + "de" + nome do cátion.
NaCl = cloreto de sódio. CaCO₃ = carbonato de cálcio.

### Óxidos
| Tipo | Característica | Exemplo |
|------|---------------|---------|
| **Ácido** | Não-metal + O₂ | CO₂, SO₂ (reage com base) |
| **Básico** | Metal + O₂ | Na₂O, CaO (reage com ácido) |
| **Anfótero** | Metal + O₂ | Al₂O₃, ZnO (reage com ambos) |
| **Neutro** | Não-metal + O₂ | CO, NO (não reage) |

**Macete Cesgranrio:** A questão dá uma reação e pede para CLASSIFICAR (ácido/base/sal/óxido) ou dar o NOME.

## 2. SOLUÇÕES (~10 questões) ← MAIS IMPORTANTE

### Fórmulas que você precisa GRAVAR

| Conceito | Fórmula |
|----------|---------|
| Concentração comum | **C = m/V** (g/L) |
| Molaridade | **M = n/V = m/(MM × V)** (mol/L) |
| Título | **τ = m₁/m** (massa soluto/massa total) |
| ppm | ppm = (m soluto / m solução) × 10⁶ |
| Diluição | **C₁V₁ = C₂V₂** (mesma para M₁V₁ = M₂V₂) |
| Mistura sem reação | C_f = (C₁V₁ + C₂V₂)/(V₁ + V₂) |

### Diluição - O que MAIS cai

**C₁V₁ = C₂V₂** — Essa fórmula resolve 90% das questões de diluição da Cesgranrio.

> Exemplo: 100 mL de HCl 0,5 mol/L são diluídos para 500 mL. Qual a nova concentração?
> C₁V₁ = C₂V₂ → 0,5 × 0,1 = C₂ × 0,5 → C₂ = 0,05/0,5 = **0,1 mol/L**

### Propriedades Coligativas
- **Tonoscopia:** pressão de vapor diminui
- **Ebulioscopia:** temperatura de ebulição aumenta
- **Crioscopia:** temperatura de congelamento diminui
- **Osmose:** passagem de solvente pela membrana

**Macete:** Toda propriedade coligativa depende do NÚMERO de partículas dissolvidas, não da natureza.

### Coloides
- Tamanho: 1-1000 nm
- Exemplos: gelatina, leite, fumaça, neblina
- **Efeito Tyndall:** dispersão da luz (feixe visível)

### Kps (Produto de Solubilidade)
- Para A₂B₃ → 2A + 3B: Kps = [A]²[B]³

**Macete Cesgranrio:** Kps sempre relaciona com solubilidade molar (s). Quanto maior o Kps, mais solúvel.

## 3. REAÇÕES E ESTEQUIOMETRIA (~6 questões)

### NOX - Número de Oxidação

**Regras:**
1. Substância simples: NOX = 0
2. Íons simples: NOX = carga
3. H: +1 (exceto hidretos: -1)
4. O: -2 (exceto peróxidos: -1)
5. Soma dos NOX em composto neutro = 0
6. Soma em íon = carga do íon

**Redox:** Quem **perde** elétrons = **oxida** (NOX aumenta) = **agente redutor**.
Quem **ganha** elétrons = **reduz** (NOX diminui) = **agente oxidante**.

### Balanceamento

**Método das tentativas (redox):**
1. Descobrir quem oxida e quem reduz
2. Calcular ΔNOX
3. Inverter os ΔNOX
4. Ajustar por tentativa

### Estequiometria - Macetes

**Fórmula mágica:** 1 mol = 6,02×10²³ partículas = 22,4 L (CNTP) = massa molar (g)

**Passo a passo:**
1. Escrever a equação BALANCEADA
2. Montar regra de três com os coeficientes
3. Converter unidades (massa, volume, mol)
4. Aplicar rendimento/pureza se houver

**Rendimento:** Se o rendimento é 80%, multiplica o resultado teórico por 0,8.
**Pureza:** Se a pureza é 90%, só 90% da massa é reagente de verdade.

## 4. MODELOS ATÔMICOS E TABELA (~1-2 questões)

### Evolução dos modelos
| Cientista | Modelo |
|-----------|--------|
| Dalton | Bola de bilhar (indivisível) |
| Thomson | Pudim de passas (elétrons na massa positiva) |
| Rutherford | Sistema solar (núcleo + elétrons) |
| Bohr | Níveis de energia |
| Schrödinger | Nuvem eletrônica (orbitais) |

### Números quânticos
| Número | Símbolo | Significado |
|--------|---------|-------------|
| Principal | n (1, 2, 3...) | Nível de energia |
| Secundário | l (0 a n-1) | Subnível (s, p, d, f) |
| Magnético | m (-l a +l) | Orbital |
| Spin | s (+1/2 ou -1/2) | Rotação do elétron |

### Tabela Periódica
- **Família 1:** Metais alcalinos (muito reativos)
- **Família 2:** Metais alcalino-terrosos
- **Família 17:** Halogênios (muito reativos)
- **Família 18:** Gases nobres (estáveis)

**Propriedades periódicas:**
| Propriedade | Cresce para |
|------------|-------------|
| Raio atômico | ← direita (diminui), ↓ baixo (aumenta) |
| Eletronegatividade | → direita, ↑ cima |
| Energia de ionização | → direita, ↑ cima |

### GABARITO COMENTADO

**Q1:** B (NaOH=base, HCl=ácido, NaCl=sal)
**Q2:** C (Fe³⁺ = íon férrico, óxido férrico)
**Q3:** C (M = m/MM·V = 40/40×0,5 = 40/20 = 2 mol/L)
**Q4:** B (M₁V₁ = M₂V₂ → 2×0,2 = M₂×0,8 → M₂ = 0,4/0,8 = 0,5)
**Q5:** E (H₂SO₄: 2(+1) + S + 4(-2) = 0 → S = 8-2 = +6)
**Q6:** A (Zn⁰ → Zn²⁺ perde elétrons = oxida)
**Q7:** C (2 mol CH₄ → 2 mol CO₂, pela proporção 1:1)
**Q8:** B (28g N₂ = 1 mol. 1 mol N₂ → 2 mol NH₃ = 34g. Com 50%: 34×0,5 = 17g)
**Q9:** C (Flúor é o elemento MAIS eletronegativo)
**Q10:** A (Kps = s² → s = √(1,8×10⁻¹⁰) = 1,34×10⁻⁵)
**Q11:** B (Cf = (0,5×0,2 + 1×0,3)/(0,5) = 0,4/0,5 = 0,8 mol/L)
**Q12:** C (CaCl₂ libera 3 íons, maior efeito coligativo)
**Q13:** D (ácido + base → sal + água = neutralização, tipo de dupla troca)
**Q14:** D (n=3 → subníveis s(2)+p(6)+d(10) = 18 elétrons)
**Q15:** C (C = 5g/0,25L = 20 g/L)
