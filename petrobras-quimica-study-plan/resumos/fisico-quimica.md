## 1. EQUILÍBRIO QUÍMICO (~5 questões) ← MUITO IMPORTANTE

### Constante de Equilíbrio (Kc)

Para aA + bB ⇌ cC + dD:
**Kc = [C]^c × [D]^d / [A]^a × [B]^b**

| Se Kc | Significado |
|-------|-------------|
| Kc > 1 | Produtos favorecidos (reação para direita) |
| Kc < 1 | Reagentes favorecidos (reação para esquerda) |
| Kc = 1 | Concentrações iguais |

**Relação Kc × Kp:** Kp = Kc(RT)^Δn (Δn = np - nr)

### Princípio de Le Chatelier
> "Quando um equilíbrio é perturbado, o sistema se desloca no sentido de minimizar a perturbação."

| Perturbação | Deslocamento |
|-------------|-------------|
| Aumenta [reagente] | → direita |
| Diminui [reagente] | ← esquerda |
| Aumenta temperatura (exotérmica) | ← esquerda |
| Aumenta temperatura (endotérmica) | → direita |
| Aumenta pressão (com gás) | Para o lado com MENOS mols |
| Catalisador | Não desloca! Só acelera |

**Macete Cesgranrio:** Se a questão tem gráfico ou tabela de concentração × tempo, o equilíbrio é quando as concentrações se estabilizam (curvas ficam horizontais).

### pH e pOH

| Fórmula | Relação |
|---------|---------|
| pH = -log[H⁺] | pH + pOH = 14 |
| pOH = -log[OH⁻] | [H⁺] × [OH⁻] = 1×10⁻¹⁴ |

**Macete:** pH < 7 = ácido. pH = 7 = neutro. pH > 7 = básico.

**Cálculo rápido de log:**
- [H⁺] = 10⁻³ → pH = 3
- [H⁺] = 5×10⁻³ → pH = 3 - log5 = 3 - 0,7 = **2,3**

### Hidrólise de Sais

| Sal formado por | Solução | Exemplo |
|----------------|---------|---------|
| Ácido forte + Base forte | Neutra (pH=7) | NaCl |
| Ácido forte + Base fraca | Ácida (pH<7) | NH₄Cl |
| Ácido fraco + Base forte | Básica (pH>7) | NaHCO₃ |
| Ácido fraco + Base fraca | Depende do Ka/Kb | NH₄CN |

### Solução-Tampão
- Resiste a variações de pH
- Geralmente: ácido fraco + sal do mesmo ácido (ou base fraca + sal)

**Equação de Henderson-Hasselbalch:**
pH = pKa + log([sal]/[ácido])

## 2. CINÉTICA QUÍMICA (~2 questões)

### Velocidade das Reações

| Fator | Efeito |
|-------|--------|
| ↑ Concentração | ↑ Velocidade |
| ↑ Temperatura | ↑↑ Velocidade (dobra a cada 10°C) |
| ↑ Superfície de contato | ↑ Velocidade |
| Catalisador | ↑ Velocidade (diminui energia de ativação) |

**Lei da Velocidade:** v = k[A]^m[B]^n
- **k:** constante de velocidade
- **m, n:** ordens de reação (determinadas experimentalmente)

**Ordem de reação:** Soma dos expoentes (m+n)
- 1ª ordem: v = k[A]
- 2ª ordem: v = k[A]² ou v = k[A][B]

**Energia de ativação:** Energia mínima para a reação ocorrer. Catalisador DIMINUI a energia de ativação.

**Macete Cesgranrio:** Questão de cinética geralmente dá uma tabela de concentração × tempo e pede para calcular a velocidade média ou identificar a ordem.

## 3. TERMOQUÍMICA (~2 questões)

### Entalpia (ΔH)

| Reação | ΔH | Significado |
|--------|-----|-------------|
| Exotérmica | ΔH < 0 | Libera calor |
| Endotérmica | ΔH > 0 | Absorve calor |

### Lei de Hess
> O ΔH de uma reação depende apenas dos estados inicial e final, não do caminho.

**Macete:** Some as equações como se fossem equações matemáticas. Cancele substâncias iguais em lados opostos.

### Entalpia de Ligação
ΔH = Σ(energia das ligações quebradas) - Σ(energia das ligações formadas)

## 4. ELETROQUÍMICA (~1 questão)

### Pilha (Célula Galvânica)

| Componente | Ocorre |
|------------|--------|
| **Ânodo** | **Oxidação** (perde e-) - polo **negativo** |
| **Cátodo** | **Redução** (ganha e-) - polo **positivo** |

**DDP (diferença de potencial):** ΔE = E⁰cátodo - E⁰ânodo
- ΔE > 0 = pilha espontânea

**Macete:** Ânodo = Oxidação = Ambas começam com A. Cátodo = Redução.

### Eletrólise
- Processo NÃO espontâneo (usa energia elétrica)
- **Ígnea:** substância fundida
- **Aquosa:** em solução aquosa

**Lei de Faraday:** m = (M × i × t) / (n × F)
- m = massa depositada
- M = massa molar
- i = corrente (A)
- t = tempo (s)
- n = número de elétrons
- F = 96500 C/mol
