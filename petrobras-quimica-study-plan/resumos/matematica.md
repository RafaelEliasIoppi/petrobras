# Resumo de Matemática - Cesgranrio (8-10 questões)

## 1. FUNÇÕES (~2 questões)

### Função Afim (1º grau) - f(x) = ax + b

| Elemento | Significado |
|----------|-------------|
| **a** (coeficiente angular) | taxa de variação, inclinação |
| **b** (coeficiente linear) | valor onde corta o eixo y |
| Raiz (zero) | x = -b/a |

**Macete:** a > 0 → crescente. a < 0 → decrescente.

### Função Quadrática (2º grau) - f(x) = ax² + bx + c

| Elemento | Fórmula |
|----------|---------|
| Vértice x | x_v = -b/(2a) |
| Vértice y | y_v = -Δ/(4a) |
| Δ (discriminante) | Δ = b² - 4ac |

**Macete:** a > 0 → concavidade para cima (valor mínimo). a < 0 → para baixo (máximo).
Δ > 0 → 2 raízes reais. Δ = 0 → 1 raiz. Δ < 0 → nenhuma raiz real.

## 2. GEOMETRIA (~2 questões)

### Geometria Plana - Áreas

| Figura | Área |
|--------|------|
| Triângulo | (b × h)/2 |
| Quadrado | L² |
| Retângulo | b × h |
| Círculo | πR² |
| Losango | (D × d)/2 |

### Geometria Espacial - Volumes

| Sólido | Volume | Área lateral |
|--------|--------|-------------|
| Cubo | a³ | 6a² |
| Paralelepípedo | a × b × c | 2(ab+ac+bc) |
| Cilindro | πR²h | 2πRh |
| Cone | πR²h/3 | πRg (g=geratriz) |
| Esfera | 4πR³/3 | 4πR² |

**Macete Cesgranrio:** Questões de geometria geralmente pedem **volume** ou **área** com dados diretos. Raramente exigem demonstrações.

---

## 3. ANÁLISE COMBINATÓRIA (~1 questão)

| Conceito | Fórmula | Quando usar |
|----------|---------|-------------|
| PFC | Multiplica as possibilidades | Etapas sucessivas |
| Permutação | Pn = n! | Ordenar todos os elementos |
| Arranjo | A(n,p) = n!/(n-p)! | Ordem importa, menos elementos |
| Combinação | C(n,p) = n!/p!(n-p)! | Ordem NÃO importa |

**Macete:** "Ordem importa?" Se sim → Arranjo. Se não → Combinação.

---

## 4. PROGRESSÕES (~0-1 questão)

### PA - Progressão Aritmética
- Termo geral: a_n = a₁ + (n-1)r
- Soma: S_n = (a₁ + a_n)n/2

### PG - Progressão Geométrica
- Termo geral: a_n = a₁ × q^(n-1)
- Soma finita: S_n = a₁(q^n - 1)/(q - 1)

---

## 5. MATRIZES (~0-1 questão)

- **Soma:** mesmas dimensões, elemento a elemento
- **Multiplicação:** A(m×n) × B(n×p) = C(m×p)
- **Determinante 2×2:** |a b; c d| = ad - bc
- **Determinante 3×3:** Regra de Sarrus

---

## 6. MATEMÁTICA FINANCEIRA (~0-1 questão)

- Juros simples: J = C × i × t | M = C(1 + it)
- Juros compostos: M = C(1 + i)^t

---

### GABARITO COMENTADO

**Q1:** C (V = 50000 - 4000t → t=8 → V = 50000 - 32000 = 18000)
**Q2:** C (x_v = -b/2a = -40/(-2) = 20)
**Q3:** C (R = 2m, A = πR² = 3,14 × 4 = 12,56)
**Q4:** D (V = πR²h = 3 × 4 × 3 = 36)
**Q5:** B (C(8,3) = 8!/3!5! = 56)
**Q6:** C (a₁₀ = 5 + 9×3 = 5 + 27 = 32)
**Q7:** B (det = 2×4 - 1×3 = 8 - 3 = 5)
**Q8:** B (20% de 500 = 0,2 × 500 = 100)
**Q9:** B (m = (6-2)/(3-1) = 4/2 = 2)
**Q10:** C (60 amostras / 3 téc × 4h = 5 amostras/téc·h. 40 amostras / 2 téc × 5 = 4h)
