## Técnicas dos Aprovados (compilado dos melhores cursinhos)

### 1. Active Recall (Recordação Ativa)
- Depois de ler um tópico, FECHE o material e tente lembrar
- Faça perguntas a si mesmo: "O que é Kps?", "Como calculo pH?"
- Use flashcards mentais

### 2. Tática dos 60% (Cesgranrio)
- Na prova, resolva PRIMEIRO as questões mais rápidas e fáceis
- Marque as que exigem cálculo para depois
- 60% de acertos = gabarito próximo (a nota de corte costuma ser 50-65%)

### 3. Matriz de Prioridades (Estratégia Concursos)
```
                  CAI MUITO │ CAI POUCO
               ┌───────────┼───────────┐
   FÁCIL P/ VC │   FOCO    │   DEPOIS  │
               ├───────────┼───────────┤
   DIFÍCIL P/V │  PRIORID. │  DEIXE P/ │
      VOCÊ     │   MÁXIMA  │  ÚLTIMO   │
               └───────────┴───────────┘
```

### 4. Caderno de Erros Inteligente
- Classifique cada erro como: (A) Desconhecimento / (B) Confusão / (C) Atenção
- Erro tipo A: revise teoria AGORA
- Erro tipo B: crie tabela comparativa
- Erro tipo C: crie gatilho mental (ex: "crase antes de verbo NUNCA")

### 5. Simulado Perfeito
- Faça em ambiente silencioso, sem interrupções
- Respeite o tempo real da prova (4h para 60 questões → 4 min/q)
- Use cartão-resposta similar ao Cesgranrio
- Corrija e analise erros no mesmo dia
