## Produto
Plataforma de estudos para Concurso Petrobras — Técnico em Química (Cesgranrio)
URL: https://rafaeleliasioppi.github.io/petrobras/

## Diferenciais Competitivos
1. **Ciclo de Estudos Ponderado** — Metodologia que distribui matérias por peso (24 slots rotativos), priorizando disciplinas mais relevantes no edital
2. **Três camadas de persistência** — Cache reativo + localStorage + servidor (nenhum dado perdido)
3. **Flashcards com revisão espaçada** — Algoritmo de repetição para memorização de longo prazo
4. **Caderno de Erros classificado** — Erros categorizados em A (desconhecimento), B (confusão) e C (atenção), com revisão D+7 e D+30
5. **Banco de Questões Cesgranrio** — 100+ questões no estilo da banca com feedback e explicação
6. **Relatório de produtividade** — Métricas de horas, ciclos concluídos, simulados e evolução
7. **Dark mode nativo** — Interface adaptável para estudo noturno
8. **Zero dependências externas** — Roda direto no GitHub Pages, sem servidor pago
